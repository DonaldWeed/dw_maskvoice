# dw_maskvoice

**dw_maskvoice** is a lightweight FiveM script that applies a realistic voice distortion
when a player is wearing a mask.

The script uses native Mumble audio submixes to slightly muffle and filter voices,
improving immersion and roleplay quality without any UI or framework dependencies.

---

## Features
- Automatic voice distortion when wearing a mask
- Uses native FiveM / Mumble submixes
- No UI
- No framework required
- Minimal performance impact

---

## Requirements
- FiveM server with **Mumble Voice** enabled

---

## Installation
1. Download or clone this repository
2. Place the folder `dw_maskvoice` into your `resources` directory
3. Add this line to your `server.cfg`: ensure dw_maskvoice
---

## Notes
- Voice distortion only applies while a mask is worn
- Removing the mask restores normal voice instantly
- Effect strength is intentionally subtle for realistic RP
